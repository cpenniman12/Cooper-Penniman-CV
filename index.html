NL2SQL MCP Server
JavaScript, Postgres, Anthropic, Pgvector

Enable LLMs to query Postgres databases using natural language.

How it works: The MCP server includes schema information in its tool descriptions. When an LLM needs database information, it sends a natural language request which gets vectorized using pgvector and matched against pre-vectorized schema tables. The top 10 semantically similar tables are sent to Claude to generate SQL from the original request, which is then executed and returned to the LLM.
